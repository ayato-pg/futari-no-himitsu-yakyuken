## 実装依頼

「2人の秘密、野球拳。」というElectronゲームを実装してください。

### プロジェクト場所
`C:\Users\PC-user\OneDrive\デスクトップ\2人の秘密、野球拳。`

### 仕様書
`GAME_SPECIFICATION.md` に完全な仕様が記載されています。

### 実装の重要ポイント

1. **ゲーム概要**
   - 25歳OL美咲と大学生主人公の野球拳ゲーム
   - じゃんけんで5勝先取でTRUE END
   - 5敗でBAD END

2. **必須4画面**
   - タイトル画面
   - 会話シーン（美咲の立ち絵のみ）
   - メインゲーム（美咲の大型立ち絵）
   - エンディング（TRUE/BAD）

3. **CSV管理（17個）**
   - 必ずBOM付きUTF-8形式で作成
   - 日本語文字化け防止
   - Excelで編集可能

4. **美咲の衣装変化**
   - レベル1: OLスーツ
   - レベル2: カジュアル服
   - レベル3: 部屋着
   - レベル4: キャミソール
   - レベル5: バスタオル

### 最初に作成するファイル

1. **package.json**
```json
{
  "name": "futari-no-himitsu-yakyuken",
  "version": "1.0.0",
  "main": "main.js",
  "scripts": {
    "start": "electron .",
    "dev": "electron . --dev"
  },
  "dependencies": {
    "electron": "^27.0.0"
  }
}
```

2. **main.js** (Electronメインプロセス)
3. **index.html** (ゲームのエントリーポイント)
4. **src/game.js** (ゲームロジック)
5. **CSVファイル17個** (assets/data/csv/配下)

### 実装順序

3. タイトル画面
